# Setup
## Virtual env
Create a virtual environment in the project folder by running in windows command promp:
```sh
$ python3 -m venv .
$ "Scripts/activate.bat"
```
## Dependencies
```sh
$ pip install flask
$ pip install pymysql
$ pip install cryptography
```
