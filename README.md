# Setup
## Virtual env
Create a virtual environment in the repository folder by running the following commands in windows command promp:
```sh
$ py -m venv .
$ "Scripts/activate.bat"
```
## Dependencies
```sh
$ pip install flask
$ pip install pymysql
$ pip install cryptography
```
